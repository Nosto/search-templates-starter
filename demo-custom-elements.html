  2. Import the custom elements in your JavaScript
  3. Ensure the SearchPageProvider context is available

Custom Event Communication

The custom elements communicate via custom events:

  • nosto-search-filters-changed - When filters are modified
  • nosto-search-filters-cleared - When all filters are cleared
  • nosto-search-sort-changed - When sort option is changed
  • nosto-search-filter-toggle - When mobile filter is toggled
  • nosto-search-product-clicked - When a product is clicked
  • nosto-search-products-loaded - When products are loaded
  • nosto-search-page-changed - When pagination changes
  • nosto-search-page-size-changed - When page size changes

JavaScript Usage

// Import the custom elements
import '@/custom-elements'

// Listen for custom events
document.addEventListener('nosto-search-filters-changed', (event) => {
    console.log('Filters changed:', event.detail)
})

// Create elements programmatically
const filters = document.createElement('nosto-search-filters')
document.body.appendChild(filters)
